（2）枚举类型

枚举类型是一种有有限可选值的数据类型，每个可选值都对应一个int值。

在Go语 言中，可以使用const关键字来定义一个枚举类型，方法如下：

type <枚举类型名> int const (

<可选值1> <枚举类型名> = iota//值为0

<可选值2> //值为1

<可选值3> //值为2

<可选值4> //值为3

)

<可选值1>由iota初始化后，后面的可选值会自动递增。<可选值1>对应整型数0,< 可选值2>对应整型数1,<可选值3>对应整型数2，以此类推。

可以使用下面的方法引用枚举类型的可选值。

<枚举类型名>.<可选值n>
